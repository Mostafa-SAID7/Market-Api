📡 API Endpoints

GET /api/products

Get all products

GET /api/products/{id}

Get product by ID

GET /api/products/GetByPriceRange/{minPrice}/{maxPrice}

Get products by price range

POST /api/products

Create a new product

PUT /api/products/{id}

Update a product

DELETE /api/products/{id}

Delete a product
