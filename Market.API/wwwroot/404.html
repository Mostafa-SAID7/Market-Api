⚠️ Common Mistakes

  • ❌ Don't try to access http://localhost:27017
    That's the MongoDB port, not a web interface
  • ✅ Use http://localhost:5000
    This is the correct API port
  • ✅ API endpoints start with /api/
    Example: /api/products
