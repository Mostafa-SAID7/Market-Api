📚 API Documentation

Complete guide to using the Market API

🚀 Getting Started

The Market API is a RESTful service for managing products. All responses are in JSON format.

Base URL:

http://localhost:5000/api

🔑 Authentication

Currently, the API does not require authentication. All endpoints are publicly accessible.

Note: Authentication will be added in future versions.

📦 Response Format

All API responses follow standard HTTP status codes:

  • ✓ 200 - Success
  • ✓ 201 - Created
  • ✓ 204 - No Content
  • ✗ 400 - Bad Request
  • ✗ 404 - Not Found
  • ✗ 500 - Server Error

📡 Endpoints Reference

GET /api/products

Description: Retrieve all products

Response:

[
  {
    "id": "69e7b3d5d118f954bb63265b",
    "name": "Laptop",
    "price": 999.99
  }
]
GET /api/products/{id}

Description: Retrieve a specific product by ID

Parameters:

  • id (string) - Product ID

Example:

GET /api/products/69e7b3d5d118f954bb63265b
GET /api/products/GetByPriceRange/{minPrice}/{maxPrice}

Description: Get products within a price range

Parameters:

  • minPrice (decimal) - Minimum price
  • maxPrice (decimal) - Maximum price

Example:

GET /api/products/GetByPriceRange/100/1000
POST /api/products

Description: Create a new product

Request Body:

{
  "name": "Laptop",
  "price": 999.99
}

Response: 201 Created with product object

PUT /api/products/{id}

Description: Update an existing product

Parameters:

  • id (string) - Product ID

Request Body:

{
  "id": "69e7b3d5d118f954bb63265b",
  "name": "Gaming Laptop",
  "price": 1299.99
}
DELETE /api/products/{id}

Description: Delete a product

Parameters:

  • id (string) - Product ID

Response: 204 No Content

💻 Code Examples

Here are some examples of how to use the API:

JavaScript (Fetch API)

// Get all products
fetch('http://localhost:5000/api/products')
  .then(response => response.json())
  .then(data => console.log(data));

// Create a product
fetch('http://localhost:5000/api/products', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ name: 'Laptop', price: 999.99 })
})
  .then(response => response.json())
  .then(data => console.log(data));

cURL

# Get all products
curl http://localhost:5000/api/products

# Create a product
curl -X POST http://localhost:5000/api/products \
  -H "Content-Type: application/json" \
  -d '{"name":"Laptop","price":999.99}'
